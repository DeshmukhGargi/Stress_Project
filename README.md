# Stress_Project
# 🧠 AI Stress Detection Web App

This project is an AI-based web application that detects stress levels from user input text using Machine Learning and Natural Language Processing (NLP).

---

## 🚀 Features

- Detects stress from user-entered text
- Uses trained ML model
- Simple and interactive web interface
- Provides basic suggestions based on stress level

---

## 🛠️ Technologies Used

- Python
- Flask
- Scikit-learn
- HTML, CSS, JavaScript
